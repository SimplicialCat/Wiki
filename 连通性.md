## 定义


称一个空间 (或 $\infty$-群胚) $n$-连通是指其 $n$ 阶及以下同伦群平凡.

在 HoTT 中, 称一个映射 $n$-连通是指其纤维 $n$-连通. 注意传统拓扑学的 $n$-连通映射在HoTT 中称为 $(n-1)$-连通映射.

## 例

| $n$  | $n$-连通空间 | $n$-连通态射 |
| ---- | -------- | -------- |
| $-2$ | -        | -        |
| $-1$ | 非空       | 满        |
| $0$  | 非空且道路连通  |          |
| $1$  | 单连通      |          |
